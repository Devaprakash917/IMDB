This project performs Exploratory Data Analysis (EDA) on IMDb movie data to uncover trends, patterns, and insights related to movie ratings, genres, popularity, and other key attributes.

Using Python data analysis techniques, the project explores how different factors influence movie ratings and success, providing valuable insights into the film industry.

Key Questions Explored:

Which genres have the highest ratings?

How do ratings vary over time?

What factors influence movie popularity?

Which movies have the highest ratings?

Is there a relationship between votes and ratings?

Key Insights:

Certain genres consistently receive higher ratings

Higher vote counts often correlate with better ratings

Movie trends vary across years

Data visualization helps identify industry patterns

Results:

The analysis highlights important trends and relationships in movie data, demonstrating how data analytics can be used to understand audience preferences and industry dynamics.
